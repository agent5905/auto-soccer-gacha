# GDD - Auto Soccer Gacha (MVP)

## Core loop
Summon -> Build squad -> Auto match -> Rewards -> Upgrade -> Repeat

## Match format (MVP)
- Team size: 5v5 (recommended for scope)
- Field: 3 zones per side (Defense / Mid / Attack)
- Match length: 60 possessions (tunable)
- Output: event log + match stats (shots, saves, possession, MVP)

## Collection
- Players have: rarity, position, base stats, 1–3 traits, 0–2 personality tags (optional MVP)
- Duplicates convert to shards to upgrade

## Progression (MVP)
- Player level: increases stats
- Simple currency: Coins + Gems
- Daily login reward

## Modes (MVP)
- PvE ladder (AI opponents)
- “Async PvP” later (fight snapshots/ghost squads)

## UX screens (MVP)
Boot/Auth -> Home -> Summon -> Collection -> Squad -> Match -> Results
